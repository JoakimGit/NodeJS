These are the commands we're using:


npm install express

Installs the express framework.

npm install -g nodemon

Installs nodemon globally.

node app.js

Runs a node app/script.

nodemon app.js

Runs the node app/script with nodemon, which restarts the application upon code changes.

npm run start-dev

Run custom scripts defined in package.json.

"scripts": {
"start-dev": "cross-env PORT=3000 nodemon app.js"
},

Example script. Uses nodemon to start the app on a specific port.
