Javascript Basics


You can save values into variables by using var, let, or const.

Var is the old way, and it didn't have scope restriction. Let has block scope. Const is similar, but also can't be reassigned.

const firstName = "Joakim";
let lastName = "Olsen";

Just like in math, variables can be used in expressions using operators like + - * / < >

In JavaScript, the = operator is used for assignment, not to check for equality. For equaility == and === are used.

const x = 5; // assigns the value 5 to x
x == "5";  // true because it only compares value, and not type (string vs number)
x === "5"; // false because it also compares type

Printing to console is done with console.log().

Strings

There are various ways to convert a string representation of a number into a real number.

These include parseInt(), Number() and the unary plus (+number).

The characters in a string can be accessed through bracket notation like string[3] which accesses the 4th letter in the string

There are many string methods available, such as indexOf(), search(), slice(), and replace().

Numbers

Javascript has only one type of number, so there's no specific float or int type. Numbers can be with or without decimals.

Adding two numbers with the + operator results in a number. Adding two strings, or a number and a string, results in string concatenation.

let x = 10 + 20; // 30
let y = 10 + "20"; // 1020

Number methods include toString(), toFixed(), valueOf(), and Number.isInteger().

Arrays

Javascript arrays can store multiple values in a single variable. The values can be of different types (numbers, strings, objects, functions).

You create an array by using square brackets.

Accessing an element in an array is done through bracket notation and the index number.

You can change the element of an array by assigning a value to an accessed element.

let fruits = ["apple", "banana", "kiwi", 3];
let myFruit = fruits[1];  // banana
fruits[0] = "pear";       // ["pear", "banana", "kiwi", 3]

There are several ways to loop through an array in JavaScript. There's the for loop like in java, the forEach loop which takes a callback function that is called on every element.

The better way is to use the more specialised functional looping methods when applicable, namely map, filter, and reduce.

const nums = [1, 3, 5, 7, 9];

for (let i = 0; i < nums.length; i++) { // using regular for loop
console.log(nums[i]);
}

nums.map(x = => console.log(x)) // using map

Example array methods: length, join(), pop(), and push().

Objects

Javascript objects look like json objects, consisting of key:value pairs, called properties. Objects are denoted by curly braces.

You access a properpy through either bracket notation using named index, or by dot notation.

let person = {
  name: "Joakim",
  age: 27,
  occupation: "student",
  nameAndAge: function() {
    return this.name + " " + this.age
  }
}; 
let myAge = person.age      // 27
let myName = person["name"] // Joakim

Objects can also have methods stored as properties, and they can be accessed with dot notation and parentheses. 

person.nameAndAge() // Joakim 27

You can add a property by saying object.propery_name = value. You can remove a property by doing delete object.property_name.

let car = {brand: "Fiat"}; // { brand: 'Fiat' }
car.weight = 800; // { brand: 'Fiat', weight: 800 }
delete car.brand; // { weight: 800 }

Functions

A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses ().

The code to be executed, by the function, is placed inside curly brackets: {}

The parentheses may include parameter names separated by commas.

Inside the function, the arguments (the parameters) behave as local variables.

function myFunction(p1, p2) {
  return p1 * p2;   // The function returns the product of p1 and p2
}

Functions can also be put into variables. You can also omit the name, and even the function keyword if you want.

const myArrowFunction = () => {
  return "Hello there from the arrow function";
}
myArrowFunction() // 'Hello there from the arrow function'
