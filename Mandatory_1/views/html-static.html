Starting the server

In order to start a server for a web app, we use the express framework. First we include the express module by using require().

The express() function from the express module then creates the application for us. Once that's done, app.listen(port) listens for a connection on a port.

const express = require("express");
const app = express();
app.listen(8080);

Serving HTML

Once the server is up, you can add routes and methods, so that you can go to different endpoints on the server.

An http method is used, together with an endpoint url and request+response objects. Serving an HTML page means including the page in the response.

app.get("/", (req, res) => {
  res.sendFile(__dirname + "/views/index.html");
});

Serving Static content

To make use of static content like images, stylesheets, and scripts, you need to tell express where to find them.

You do this with the .static(folderName) function in express.

app.use(express.static('public'));
